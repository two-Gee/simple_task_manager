# Starten 

## Client 
1. Zu Client Ordner navigieren 
2. `npm i` ausführen
3. `npm run dev` ausführen

## Server 
1. In neuem Terminal zu Server Ordner navigieren 
2. `npm i` ausführen
3. `npm run dev` ausführen
4. `http://localhost:5173/` in Browser öffnen
